## Voyah Radio v1.0.3

### What's new

- **Slider sizing fix** — Fixed a bug where the station grid size slider was not applying proper sizing to the Saved Radios row. Saved station cards now get the correct height and padding so the station name and second line (e.g. "Arabic" or "Lebanon") are no longer cut off, especially when the slider is at maximum size.
